# EagleEyes Frontend Showcase

This repository highlights my personal frontend contributions to the **EagleEyes** capstone project.  
The project was completed across **two semesters** as part of our university program.

## 📚 Project Overview
During the **first semester**, our team focused heavily on planning and documentation:
- Software Development Life Cycle (SDLC)
- Software Requirements Specification (SRS)
- Workbench setup and design planning
- UI wireframes and early frontend coding before the semester ended

We continued development **through the summer and into the second semester**, expanding the UI, refining the workflow, and integrating real data for testing.  
This repo contains only my **frontend showcase**, representing the parts I personally built and designed.

## 🧠 My Role
- Frontend Lead Developer  
- Responsible for UI/UX design, component structure, and integration with backend APIs  
- Focused on building a clean, responsive interface using React and TypeScript

## 🛠️ Tech Stack
- React  
- TypeScript  
- Expo Router  
- CSS / Tailwind
